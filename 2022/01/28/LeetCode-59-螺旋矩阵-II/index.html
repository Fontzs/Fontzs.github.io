59.螺旋矩阵 II

题目

Given a positive integer n, generate an n x n matrix filled with elements from 1 to n² in spiral order.

Example 1:

Input: n = 3 Output: [[1,2,3],[8,9,4],[7,6,5]]

Example 2:

Input: n = 1 Output: [[1]]

Constraints:

  • 1 <= n <= 20

题解

这道题并没有考察算法，难点在于寻找螺旋数组生成的规律。

要寻找规律可以模拟生成过程走一遍

此图反映了螺旋数组生成的简单过程，这张图片有很明显的对称特点，而通过这一特点可以提取出生成规律：

  • 若将循环一周作为一个周期，则每个二维数组有 n/2 个周期
  • 每个周期可以拆解为四个部分：从左到右，从上到下，从右到左，从下到上
  • 建立x，y直角坐标系（右手系），新周期的起始位置的横纵坐标与上一周期结束位置的横纵坐标分别相差一
